git installation
stages:
U- untracked 
A- added or staged
C- commited

Must know commands-
git status -s -> know current status of unstaged and stageg files
git log --oneline-> know current status of saved points

How to manage your own projects?
1.Make git available in your project
2.Making a checkpoint or saved point
i.adding files
ii.staging them
iii.commiting them
Going back to some previous saved point
logging everything
reverting back to previous saved point
