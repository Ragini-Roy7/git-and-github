#  Learn Git & GitHub – Beginner Friendly

A **clean, beginner-focused repository** to help students and first-time open-source contributors learn **Git fundamentals and the GitHub workflow** the way it is used in real projects like **GSSoC, Hacktoberfest, and internships**.

> 🎯 **Goal**: Build strong fundamentals so contributors can confidently create branches, commits, and pull requests **without fear of breaking anything**.

---

## 📌 Who is this repository for?

This repository is designed for:

* 🌱 **Beginners / Freshers** learning Git & GitHub
* 🎓 Students preparing for **GSSoC / Hacktoberfest**
* 👩‍💻 First-time open-source contributors
* Anyone who knows basic coding but struggles with collaboration workflows

No advanced coding knowledge required.

---

## 🧠 What you will learn

By the end of this repo, you will be able to:

* Understand **what Git is vs GitHub**
* Track file changes using Git
* Create and switch branches
* Write meaningful commit messages
* Push code to GitHub
* Open a **clean, mentor-friendly Pull Request (PR)**
* Follow contribution guidelines

## 🛠️ How to get started

### 1️⃣ Fork this repository

Click the **Fork** button on the top-right corner of this repo.

### 2️⃣ Clone your fork

```bash
git clone https://github.com/your-username/learn-git-github.git
cd learn-git-github
```

### 3️⃣ Create a new branch

```bash
git checkout -b beginner/your-name
```

### 4️⃣ Make your changes

* Improve documentation
* Fix typos
* Add examples
* Add beginner-friendly explanations

### 5️⃣ Commit your changes

```bash
git add .
git commit -m "docs: improve explanation of git commit"
```

### 6️⃣ Push and create PR

```bash
git push origin beginner/your-name
```

Open a Pull Request and explain **what you changed and why**.
